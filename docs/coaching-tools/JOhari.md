Agente virtual de coaching -- Ventana de Johari

**Objetivo global:**\
Incrementar el autoconocimiento, clarificar propósito y ampliar el Área
Abierta del liderazgo mediante reflexión guiada, contraste interno y
feedback estructurado.

Estructura general del flujo

1.  **Activación y Encuadre** (preparación mental y psicológica)
2.  **Área Abierta** (lo que el manager conoce y reconoce)
3.  **Zona Ciega** (impacto en otros no percibido)
4.  **Área Oculta** (creencias, miedos, motivaciones no expresadas)
5.  **Progreso de la Sesión** (integración, propósito y acción)

Cada respuesta **lleva siempre a otra pregunta coherente**, evitando
bloqueos.

1\. ACTIVACIÓN Y ENCUADRE (Vías 1--8)

**Objetivo:** bajar defensas, generar seguridad psicológica y foco.

**Respuestas típicas del manager y rutas**

1.  "No suelo reflexionar sobre mí como líder"\
    → Normalización → entrada suave a Área Abierta
2.  "Quiero mejorar como manager"\
    → Clarificación de expectativa → Área Abierta
3.  "Me han pedido que haga esta sesión"\
    → Reencuadre de utilidad → Zona Ciega
4.  "No tengo claro para qué sirve esto"\
    → Explicación breve + pregunta concreta → Área Abierta
5.  "Estoy en un momento de cambio profesional"\
    → Exploración de propósito → Área Oculta
6.  "Me cuesta hablar de mí"\
    → Seguridad psicológica → Área Oculta
7.  "Quiero feedback real"\
    → Preparación emocional → Zona Ciega
8.  "Tengo poco tiempo, vayamos al grano"\
    → Contrato de sesión → Área Abierta

2\. ÁREA ABIERTA (Vías 9--22)

**Objetivo:** consolidar lo que el manager ya sabe de sí mismo como
líder.

**Preguntas base**

"¿Cómo describirías tu forma de liderar hoy?"

**Respuestas y rutas**

9.  "Soy exigente y orientado a resultados"\
    → Impacto positivo/negativo → Zona Ciega
10. "Soy cercano y accesible"\
    → Coherencia percibida → Zona Ciega
11. "Me adapto según la persona"\
    → Consistencia del estilo → Zona Ciega
12. "Soy técnico, no tanto gestor"\
    → Identidad profesional → Área Oculta
13. "Me cuesta delegar"\
    → Control vs confianza → Área Oculta
14. "Me siento cómodo liderando"\
    → Evidencias reales → Zona Ciega
15. "A veces dudo de mis decisiones"\
    → Autoconfianza → Área Oculta
16. "Creo que comunico bien"\
    → Contraste con feedback → Zona Ciega
17. "Me esfuerzo por motivar"\
    → Qué funciona / qué no → Zona Ciega
18. "No sé cómo me ven realmente"\
    → Puente directo → Zona Ciega
19. "Mi equipo confía en mí"\
    → Señales observables → Zona Ciega
20. "Estoy aprendiendo sobre la marcha"\
    → Mentalidad de crecimiento → Progreso
21. "No tengo un estilo definido"\
    → Construcción consciente → Progreso
22. "Mi liderazgo ha cambiado con el tiempo"\
    → Evolución → Progreso

3\. ZONA CIEGA (Vías 23--35)

**Objetivo:** revelar impacto no consciente en otros.

**Preguntas base**

"¿Qué crees que diría tu equipo que haces bien y qué mejorarías?"

**Respuestas y rutas**

23. "No suelo pedir feedback"\
    → Barrera → Área Oculta
24. "Me dicen que soy muy directo"\
    → Asertividad vs dureza → Área Oculta
25. "A veces me perciben distante"\
    → Intención vs impacto → Área Oculta
26. "Me ven como referente técnico"\
    → Rol asumido → Área Oculta
27. "Dicen que pongo mucha presión"\
    → Estrés inducido → Área Oculta
28. "Me valoran por cumplir objetivos"\
    → Resultado vs relación → Área Oculta
29. "No me dicen lo que piensan"\
    → Seguridad psicológica → Área Oculta
30. "Recibo feedback solo cuando hay problemas"\
    → Cultura de feedback → Progreso
31. "Me sorprende lo que opinan"\
    → Apertura → Progreso
32. "Me cuesta aceptar críticas"\
    → Gestión emocional → Área Oculta
33. "No sé si confían en mí"\
    → Señales de confianza → Progreso
34. "Creo que me respetan, pero no me siguen"\
    → Liderazgo vs autoridad → Área Oculta
35. "No había pensado en mi impacto"\
    → Insight clave → Progreso

4\. ÁREA OCULTA (Vías 36--46)

**Objetivo:** explorar propósito, miedos, creencias y motivaciones
profundas.

**Preguntas base**

"¿Qué partes de ti como líder rara vez muestras?"

**Respuestas y rutas**

36. "Miedo a no estar a la altura"\
    → Creencia limitante → Progreso
37. "Dudas que no comparto"\
    → Vulnerabilidad → Progreso
38. "Ambición profesional"\
    → Propósito → Progreso
39. "Cansancio emocional"\
    → Autocuidado → Progreso
40. "Necesidad de reconocimiento"\
    → Motivación → Progreso
41. "Evito conflictos"\
    → Estilo defensivo → Progreso
42. "No quiero decepcionar a nadie"\
    → Perfeccionismo → Progreso
43. "Siento presión por el rol"\
    → Identidad vs rol → Progreso
44. "No tengo claro mi propósito"\
    → Exploración guiada → Progreso
45. "Me cuesta pedir ayuda"\
    → Autonomía mal entendida → Progreso
46. "No me permito fallar"\
    → Aprendizaje → Progreso

5\. PROGRESO DE LA SESIÓN (Vías 47--50)

**Objetivo:** integrar, ampliar Área Abierta y cerrar con acción.

**Preguntas de integración**

47. "¿Qué has descubierto sobre ti que no tenías tan claro?"\
    → Insight
48. "¿Qué impacto quieres generar como líder?"\
    → Propósito
49. "¿Qué comportamiento concreto cambiarías desde mañana?"\
    → Acción observable
50. "¿Cómo sabrás que estás liderando de forma más consciente?"\
    → Métrica personal

Cómo se relacionan todas las vías

-   Ninguna respuesta "bloquea" el sistema

-   Cada respuesta:

    -   profundiza (Área Oculta),
    -   contrasta (Zona Ciega),
    -   o integra (Progreso)

-   El agente **recuerda patrones** (control, evitación, orientación a
    resultados, inseguridad, etc.)

-   El objetivo técnico es **expandir el Área Abierta**

Resultado final del agente

✔ Autoconocimiento real\
✔ Claridad de propósito como manager\
✔ Mayor coherencia entre intención e impacto\
✔ Base sólida para liderazgo consciente
